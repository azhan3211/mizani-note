# mizani-note
Clean Architecture + MVVM + Room + Coroutine + Flow + Koin Injection


Folder structure:
- data
  - dao
  - dto
  - entities
  - repository
- domain
  - mapper
  - repository (implementation)
- presentation
  - adapter
  - base
  - dialog
  - fragments
- utils

Feature
- CRUD operation for note
- Set reminder
- Repeatable reminder
- Choose background color for note
- Categorized note
- Search and see note from other date
